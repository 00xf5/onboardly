# Onboardly

Smart client onboarding tool. Standardize workflows, automate communication, and track progress effortlessly.

## Project info

This project is built with React, Vite, and Tailwind CSS.

## Getting Started

Follow these steps to run the project locally:

```sh
# Step 1: Clone the repository
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory
cd onboardly

# Step 3: Install dependencies
npm i

# Step 4: Start development server
npm run dev
```

## Technologies

- **Vite**: Modern frontend tooling
- **TypeScript**: Static typing for JavaScript
- **React**: UI library
- **Shadcn/UI**: High-quality UI components
- **Tailwind CSS**: Utility-first CSS framework
- **Lucide React**: Icon library
- **React Query**: Server-state management
- **Sonner**: Toast notifications

## Deployment

Deploy with ease using production-ready builds:
```sh
npm run build
```
